Internet Of Things (IoT)

The Internet of Things (IoT) is the network of real-world objects, embedded with sensors and software, that are able to transmit and receive data over the Internet.

The IoT includes objects like smart fridges, self-driving cars, smart fitness watches, smart home security systems and many more.

These devices are constantly processing the data they receive from their sensors, and can transmit this data over the Internet to various locations, such as the smart object's users or a server for further processing.

For example, smart home security systems are always processing the images taken by the camera by using machine learning to detect any suspicious activity in its view. Because it is connected to the Internet, the security system can automatically send a notification to the user (the home owner) to alert them of the suspicious activity.

This allows for the home owner to respond much quicker than if the video was simply stored in secondary storage to be reviewed upon their return, thereby greatly improving the security of the user's home.
